# SERVICE_ICONS.md

Service Icons are tiny illustrations.

They are not UI icons.

They are not SVGs.

They belong to the same neighbourhood as the Door, the House, the Robot and The Neighbour.

Every service icon should feel handcrafted, warm and human.

The goal is not to represent software.

The goal is to represent the feeling behind each action.

A WhatsApp icon should feel like:
"I'm here if you need me."

A Menu icon should feel like:
"Come in and have a look around."

A Website icon should feel like:
"Your little place on the Internet."


# Create a standalone service icon for Vecina Digital.

## STYLE

Hand-painted watercolor illustration.

Storybook aesthetic.

Soft brush textures.

Rounded organic shapes.

Warm terracotta.

Olive greens.

Warm creams.

Honey wood.

Subtle paper-like paint texture inside the illustration only.

Same artistic language as:

• The Door
• The House
• The Robot
• The Neighbour

The icon should immediately feel like it belongs to the same illustrated neighbourhood.

## COMPOSITION

Single object.

Centered.

No frame.

No badge.

No circle.

No container.

No speech bubble.

Small botanical decoration integrated naturally.

Readable at very small sizes.

## BACKGROUND

Transparent PNG.

Alpha transparency.

Absolutely NO background.

No paper.

No canvas.

No beige rectangle.

No shadow outside the illustration.

Only the icon.

## MOOD

Friendly.

Human.

Cozy.

Crafted with love.

Like a tiny illustration from a children's storybook.
