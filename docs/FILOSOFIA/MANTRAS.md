# MANTRAS

> *Las pequeñas decisiones construyen grandes proyectos.*

---

- Soy su vecina.

- Las personas antes que la tecnología.

- Escuchar primero. Explicar siempre.

- Nunca vender lo que un negocio no necesita.

- No vendo páginas web. Ayudo a negocios.

- La confianza vale más que el diseño.

- Una web bonita no sirve si el cliente no la entiende.

- Cada negocio tiene una historia que merece ser contada.

- La sencillez siempre gana.

- Internet también puede ser cercano.

- Que Internet aprenda a hablar el idioma de los pueblos.

- Abrimos puertas para que el talento de nuestros pueblos pueda ser encontrado.

- La tecnología es el medio. La relación es el verdadero producto.

- Si tenemos dudas, elegimos siempre el camino que más ayude a las personas.

- Dejamos cada negocio un poco mejor de como lo encontramos.
